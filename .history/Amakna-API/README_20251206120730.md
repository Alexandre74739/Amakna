# Configuration

2. Installer les dépendances
npm install

3. Configurer l’API

Crée un fichier .env à la racine si nécessaire.

Exemple de variables d’environnement pour ton API :

API_URL=https://ton-api.com
API_KEY=ta_cle_api
PORT=3000


Ton code Node doit utiliser ces variables via process.env.API_URL et process.env.API_KEY.

4. Lancer le serveur
Option 1 : Avec Node
node server.js

Option 2 : Avec npm

Vérifie que ton package.json contient un script start :

"scripts": {
    "start": "node server.js"
}


Puis lance :

npm run start
